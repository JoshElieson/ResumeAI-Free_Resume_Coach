[ResumeAI](https://your-live-site-link.com)  

AI-powered resume analysis platform that provides intelligent scoring, inline feedback, resume comparisons, and career-focused AI guidance.  

## Features

📄 Upload and analyze PDF resumes  
📊 AI-powered resume scoring  
📝 Inline strengths, weaknesses, and suggestions  
🤖 Resume & career chatbot  
📚 Resume scan history and comparisons  
⚙ Advanced targeting for roles, companies, and industries  
🎨 Modern responsive dark-mode UI
